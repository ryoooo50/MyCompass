# My Compass

大阪大学をイメージした白・濃紺基調の個人用ダッシュボードです。

> [!IMPORTANT]
> 現在表示している金額、タスク、研究進捗、勤務時間、予定はすべてUI確認用のモックデータです。実際の個人データとは接続していません。

## 開発

VS Codeでこのフォルダを開き、`index.html` をLive Serverで開きます。

追加パッケージを使わないため、次の方法でも起動できます。

```bash
python3 -m http.server 8000
```

その後、`http://localhost:8000` を開いてください。

## 現在の機能

- 財務・タスク・研究・アルバイト・予定の全体表示
- タスク完了状態の端末内保存
- Obsidian / Notion / Calendar / Drive / GitHubへの導線
- Obsidian Vault名とNotion URLの端末内保存
- スマートフォン表示

表示中の数値と予定は、画面設計を確認するためのデモデータです。

## データ方針

- 現在はブラウザの `localStorage` のみを使用
- 外部サービスへのデータ送信なし
- APIキーやNotionトークンをフロントエンドへ保存しない
- 将来のNotion同期はローカルバックエンド経由で実装
